Interpretation

The pattern shows relative proportions, not a literal grid:

  • Square 1: Size = 1 unit
  • Square 2: Size = 2 units (2× Square 1)
  • Square 3: Size = 3 units (3× Square 1)
  • Square 5: Size = 5 units (5× Square 1)

The arrangement creates a rectangle with aspect ratio approximately 8:5.

CSS Grid Solution

Use fractional units (fr) to create proportional sizes:

.fibonacci-grid { display: grid; grid-template-columns: 1fr 2fr 5fr; /* 1:2:5 ratio */ grid-template-rows: 1fr 1fr 3fr 3fr 3fr; /* 1:1:3:3:3 = total 11 */ gap: 4px; aspect-ratio: 8/5; /* Approximately matches pattern */ max-width: 400px; } /* Square 1 (first) - top-left */ .fib-square[data-index="0"] { grid-column: 1; grid-row: 1; } /* Square 2 - middle-top, 2 units tall */ .fib-square[data-index="2"] { grid-column: 2; grid-row: 1 / span 2; } /* Square 5 - right side, 5 units tall */ .fib-square[data-index="4"] { grid-column: 3; grid-row: 1 / span 5; } /* Square 1 (second) - left, row 2 */ .fib-square[data-index="1"] { grid-column: 1; grid-row: 2; } /* Square 3 - bottom-left, spans columns 1-2, rows 3-5 */ .fib-square[data-index="3"] { grid-column: 1 / span 2; grid-row: 3 / span 3; }

Note: This creates a non-square container (8:5 aspect ratio) that matches the pattern's proportions.
